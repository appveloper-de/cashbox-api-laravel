# cashbox-api

This is the api for the team cashbox project. It provides the HTTP (and CLI) endpoints for the cashbox application.

## Prerequisites

The easiest way to run this is to use docker (-compose).

## Installation

Just run
```
docker-compose up
```

and enjoy.
